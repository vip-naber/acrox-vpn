# 📡 Acrox VPN Ultra v2.0

Acrox Systems Labs tarafından geliştirilen, Google Chrome için açık kaynaklı, yüksek hızlı ve güvenli siber tünelleme ve proxy uzantısı. 22 farklı global veri merkezi omurgasıyla kesintisiz siber anonimlik sağlar.

![Acrox VPN Logo](acrox.png)

## 🚀 Özellikler
* **22 Global Lokasyon:** Dünya çapında internet altyapısı en güçlü veri merkezleri.
* **Sıfır Veri Logu:** Tarayıcı düzeyinde veri saklamayan siber mimari.
* **Ultra Hafif:** Arka planda sistemi ve RAM'i yormayan temiz JavaScript motoru.
* **Geliştirilebilir İskelet:** Açık kaynak kod yapısıyla tamamen modüle edilebilir.

## 🛠️ Kurulum Kılavuzu (Uzantıyı Chrome'a Ekleme)

Uzantıyı bilgisayarınıza kurup anında kullanmak için şu 3 basit siber adımı takip edin:

1. Bu GitHub sayfasındaki yeşil **`Code`** butonuna basın ve **`Download ZIP`** seçeneğine tıklayarak dosyaları bilgisayarınıza indirin. ZIP dosyasını bir klasöre çıkartın.
2. Google Chrome tarayıcınızda yeni bir sekme açıp şu gizli adrese gidin: 
   `chrome://extensions/`
3. Sağ üst köşede bulunan **"Geliştirici Modu" (Developer Mode)** çentiğini aktif hale getirin.
4. Sol üstte beliren **"Paketlenmemiş öğe yükle" (Load unpacked)** butonuna tıklayın ve ZIP'ten çıkarttığınız klasörü seçin!
